About StudentVC

StudentVC is a cutting-edge verifiable credentials platform that implements the W3C VC Data Model 2.0 with advanced cryptographic features.

Key Features:

  • BBS+ signatures for selective disclosure
  • X.509 certificate integration with DIDs
  • OpenID for Verifiable Credentials (OID4VC)
  • Cross-platform mobile wallet support
  • Multi-tenant architecture for universities

Quick Help

For Students

Use the Issuer page to generate your student credential. Scan the QR code with the StudentVC mobile app to store your credential securely.

For Verifiers

Use the Verifier page to create presentation requests and verify student credentials with selective disclosure capabilities.

System Status

Check the Status page for credential validation and system health monitoring.
